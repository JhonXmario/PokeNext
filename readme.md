# PokéNext | Pokédex Application

PokéNext is a modern, high-performance web application built with **Next.js 14** and **TypeScript** to explore and analyze Pokémon data. The project integrates the public **PokéAPI** to provide full details of all Pokémon across generations (I-IX) with a responsive, mobile-first interface.

---

## Key Features

* **Complete Catalog:** Explore over 1,000 Pokémon with progressive, optimized loading.
* **Advanced Filters:** Combine filters by generation (I-IX), type, game association, and text search.
* **Detailed Stats & Evolutions:** Visual statistics charts, physical characteristics, and dynamic evolution chains.
* **Dark Mode:** Native light/dark mode support that respects system preferences.
* **Responsive Design:** Mobile-first design crafted with Tailwind CSS v4 and shadcn/ui.

## Tech Stack

* **Framework:** Next.js 14+ (App Router)
* **Language:** TypeScript
* **CSS Framework:** Tailwind CSS v4
* **UI Components:** shadcn/ui

## Getting Started

### Prerequisites

* Node.js 18+
* pnpm or npm

### Installation

Clone the repository:

```bash
git clone https://github.com/JhonXmario/PokeNext.git
```

Install dependencies:

```bash
pnpm install
# or
npm install
```

Run the development server:

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
